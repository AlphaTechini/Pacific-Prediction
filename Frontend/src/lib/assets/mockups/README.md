# Frontend/src/lib/assets/mockups

## Purpose

I use this folder for stored mockup screenshots and concept references for the frontend surfaces.

## Architectural Decisions And Tradeoffs

- I keep these references separate from live code so design assets do not get confused with implemented behavior.
- They are useful for visual direction, but they are not authoritative product documentation.
